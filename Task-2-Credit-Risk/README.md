# Task 2: Credit Risk Prediction

## Project Objective
To develop a supervised binary classification machine learning pipeline that evaluates historic loan application profiles and predicts the likelihood of loan approval or default.

## Approach
- Handled missing values using median/mode imputation strategies.
- Applied logarithmic feature transformation (`np.log1p`) to skewed income attributes.
- Implemented and evaluated Logistic Regression, Decision Trees, and Random Forest Classifiers using 5-Fold Stratified Cross-Validation.

## Results & Champion Model
- **Champion Model:** Random Forest Classifier
- Complete visual evaluations and accuracy metrics are included in the generated PDF report.
